# 🛒 AI Shopping Chatbot

An AI-powered shopping assistant built using **Spring Boot, MySQL, and a modern UI** that helps users find products based on their needs and budget.

---

## 🚀 Features

* 🤖 Chat-based product recommendation
* 💰 Budget filtering (e.g., under ₹20,000)
* 📱 Product listing with price & details
* 🔐 User authentication (login/register)
* ⚡ Fast backend using Spring Boot
* 🗄️ MySQL database integration

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Java (Spring Boot)
* **Database:** MySQL
* **Build Tool:** Maven

---

## ▶️ How to Run

1. Start MySQL server
2. Update database credentials in `application.properties`
3. Run project using:

```
mvn spring-boot:run
```

4. Open browser:

```
http://localhost:8080
```

---

## 💡 Example Queries

* "I want a phone"
* "Show me phones under 20000"
* "Best phones under 50000"

---

## 📸 Output

Displays product recommendations with:

* Name
* Price
* Description
